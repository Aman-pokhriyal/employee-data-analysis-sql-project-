# employee-data-analysis-sql-project-
This project focuses on analyzing data using SQl queries to extract valuable insights about workspaces demographics , salaries and age distribution . the datasheet contains key employee information such as employee ID , name, gender , salary , and date_of_birth, . Various SQL queries are applied to perform statistical and comparative analysis .
Tools and Technologies : Database :- PostgreySQL , Language :- SQL , Function used :- count(),AVG(),Min(),Max(),Group By , Age(), Extract(), Corr().
Outcomes :- This project provide a detailed understanding of employee demographics and compensation trends , helping organization make data - driven decisions related to workfoeces planing and salary structuring.
